# NICE Platform - Lufthansa Technik

A comprehensive sales and project management platform for Lufthansa Technik's Network In-flight Content & Entertainment (NICE) system.

![Lufthansa Technik](https://img.shields.io/badge/Lufthansa_Technik-NICE_Platform-f0ab00?style=for-the-badge)

## Features

- **Product Catalog** - Complete overview of all NICE system components (hardware & software)
- **Project Management** - Track client implementations across Bombardier, Gulfstream, Dassault, and more
- **Project Configurator** - Build custom system configurations with automatic dependency resolution
- **Development Roadmap** - Interactive Gantt chart with scheduling details
- **Team Portal** - Team member contributions tracking and project assignments
- **Cost Calculator** - Automatic pricing with volume discounts and payment schedules
- **PDF Export** - Generate professional project quotes

## Technology

- React 18
- Single-file deployment (no build process required)
- Responsive design
- Modern UI with Lufthansa Technik branding
